# For teachers

## Collecting choices

Pick one of three ways to gather student choices on a timetable:

- **Fill in here** — type students and their ranked choices directly.
- **Upload CSV** — columns `student_name, choice1…choiceN, backup1…`.
- **Student emails** — paste a list; students sign up (or use a magic link) and fill
  in their own choices. Track who's done it from the progress bar.

Entries have a status: **awaiting student**, **draft**, **submitted**, or
**teacher entered**. You can edit any student's choices, or revert a submission back
to draft to reopen it.

## Processing

When choices are in, open **Processing**. Drag classes into blocks to pin them, or
leave them in *Automatic placement* to let the solver decide. Run it, review the block
layout and per-student assignments, then **finalise** the solution. You can optionally
let students reassign into classes that still have free space.
