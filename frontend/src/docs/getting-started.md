# Getting started

Blockwise helps schools build option blocks: collect each student's ranked subject
choices, then solve which subjects sit in which block and which class each student
lands in — maximising how many top choices are honoured.

## The basics

1. **Create a timetable** from your dashboard.
2. **Add subjects**, each with one or more classes (and their sizes).
3. **Set the rules**: how many blocks, how many choices each student ranks, how many
   backups, and any constraints (e.g. "choice 1 must be History or Geography").
4. **Collect choices** — type them in, upload a CSV, or invite students by email.
5. **Run processing** to generate optimal blocks, then review and finalise a solution.

> Tip: you can switch between the teacher and student views at any time using the
> pill in the top-right of the header.
